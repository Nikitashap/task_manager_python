# task_manager_python
Простой сервис для управления задачами. Фронт представляет собой две html страницы. Бэкенд написан на Python с использованием FastAPI. Внутри реализованы функции регистрации пользователя, входа в свой аккаунт, управление собсвенными задачами. Можно добавлять новые задачи, обновлять задачи и удалять задачи. Общение между фронтом и бэком происхожит при помощи http и REST. БД - Postgres.
